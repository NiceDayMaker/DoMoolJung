두물정은

"한글로 프로그래밍을 할 수 있을까?"

라는 궁금증에서 출발하여 만들어진 언어입니다.
두물정의 이름은 'ㄷ'과 '井 우물 정'을 합쳐 만들어졌습니다.

두물정의 사용법

두물정은 오직 한글, 숫자, 기호만 구문으로 사용 가능한 언어입니다.

가 나 다 1 2 3 # ! ? ...

위와 같이 사용하며 이때 결과값의 자료형은 첫번째 값의 자료형를 따릅니다. 정수형과 실수형은 연산 시 암시적 형변환이 됩니다. 실수형을 정수형으로 변환할 때 소수점은 버려집니다. '나머지'연산은 몫이 정수일 때 나머지 값을 반환하며 첫번째 변수와 두번째 변수가 모두 정수형 일때만 정수형을 반환하고 아니라면 실수형을 반환합니다.
