ㄷ井 두물정 SDK 1.0 가이드

두물정은

"한글로 프로그래밍을 할 수 있을까?"

라는 궁금증에서 출발하여 만들어진 언어입니다.
두물정의 이름은 'ㄷ'과 '井 우물 정'을 합쳐 만들어졌습니다.

두물정의 사용법

두물정은 오직 한글, 숫자, 기호만 구문으로 사용 가능한 언어입니다.

가 나 다 1 2 3 # ! ? ...

두물정의 UI

다음은 두물정의 기본 화면입니다.
㉮ 소스파일(.dmj)을 엽니다.
㉯ 작성한 코드를 저장(.dmj)합니다.
㉰ 도움말 창을 엽니다.
㉱ 코드의 줄 번호를 표시합니다.
㉲ 여기에 코드를 작성합니다.
㉳ 작성한 코드를 컴파일 및 실행합니다.
㉴ 열려있는 문서의 주소를 표시합니다.

단축키

코드 저장 : Ctrl + S
코드 열기 : Ctrl + O
코드 실행 : Ctrl + P
실행 중지 : Esc

두물정의 자료형

두물정의 자료형은 실수, 정수, 문자열이 있습니다.

연산과 판단

자료형 연산하기

두물정의 연산자는 '더하기', '뺴기', '곱하기', '나누기', '나머지'가 있습니다. 연산식을 사용할 떄에는 접두사로 '식'을 붙여야 합니다.
주석 식 [첫번째변수] [연산자] [두번째변수]

출력 식 15.0 곱하기 3
출력 "\n"
출력 식 15 나누기 4.0
출력 "\n"
출력 식 5 나머지 2.2
---------------실행 결과---------------
45.000000
3
0.600000
위와 같이 사용하며 이때 결과값의 자료형은 첫번째 변수를 따릅니다. 정수형과 실수형은 연산 시 암시적 형변환이 됩니다. 실수형을 정수형으로 변환할 때 소수점은 버려집니다. '나머지'연산은 몫이 정수일 때 나머지 값을 반환하며 첫번째 변수와 두번째 변수가 모두 정수형 일때만 정수형을 반환하고 아니라면 실수형을 반환합니다.

문자열의 연산 규칙

문자열의 연산 규칙은 다른 자료형과 다릅니다.

덧셈에서 두번째 자료형은 문자열형으로 암시적 형변환이 되어 첫번째 자료형의 뒤에 이어 붙여집니다.
            
주석 식 [문자열형] 더하기 [아무 자료형]

출력 식 "숫자 : " 더하기 3

---------------실행 결과---------------
숫자 : 3
뺄셈에서 두번째 자료형은 문자열형으로 암시적 형변환이 됩니다. 이때 소수점은 버려집니다. 그 후 첫번째 문자열에서 두번째 문자열에 포함되어 있는 모든 문자를 제거합니다.
주석 식 [문자열형] 뺴기 [아무 자료형]

출력 식 "이곳은 한글 세계 한글 한글" 빼기 "곳은 한글"
출력 식 "123454321" 빼기 234

---------------실행 결과---------------
이세계151
곱셈에서는 문자열을 정수만큼 반복하여 반환합니다.
주석 식 [문자열형] 곱하기 [정수형]

출력 식 "가나다" 곱하기 3

---------------실행 결과---------------
가나다가나다가나다

자료형 판단하기

두물정은 판단식에서 판단 연산자로 '같다', '다르다', '크다', '작다'를 사용합니다. 또한 사용을 위해 판단식의 접두사로 '논리'를 붙입니다.
주석 논리 [첫번째 변수] 조사 [두번째 변수] 조사 [판단 연산자]
'같다'과 '다르다'는 두 변수의 자료형이 같은지와 값이 같은지 비교합니다. 반환값은 정수형 1 또는 0입니다.
출력 논리 1 과 1.0 이 같다
출력 "\n"
문자열 가 는 "가나다"
출력 논리 가 와 "거너더" 가 다르다

---------------실행 결과---------------
0
1
'크다'와 '작다'는 두 변수의 크기를 비교합니다. 실수형과 정수형 끼리만 비교가 가능하며 첫번째 변수의 자료형으로 암시적 형변환이 됩니다. 이때 소수점은 버려집니다.
출력 논리 1 이 1.5 보다 크다

주석 논리 "가갸" 가 "나냐" 보다 작다 => 숫자형 외의 자료형은 오류 발생
주석 논리 "가나다" 가 10 보다 작다 => 숫자형 외의 자료형은 오류 발생

---------------실행 결과---------------
0

연산식과 판단식의 중첩

두물정은 연산식과 판단식을 중첩하여 사용할 수 있습니다. 하나의 "식" 또는 "논리" 구문은 괄호와 같은 효과를 냅니다.
주석 [(1 + 3) * 5]는 다음과 같이 나타낼 수 있습니다.
출력 식 식 1 더하기 3 곱하기 5

출력 "\n"

주석 판단식도 중첩하여 쓸 수 있습니다.
출력 논리 식 식 1 더하기 3 곱하기 5 와 20 이 같다

---------------실행 결과---------------

1
이를 이용해 Or, And, Not도 구현이 가능합니다.
주석 Or
출력 논리 식 논리 1 과 1 이 같다 더하기 논리 2 와 2 가 다르다 가 0 보다 크다

출력 "\n"

주석 And
출력 논리 식 논리 1 과 1 이 같다 더하기 논리 2 와 2 가 같다 가 2 와 같다

출력 "\n"

주석 Not
출력 논리 논리 1 과 1 이 다르다 와 0 이 같다

1

두물정의 실행 흐름 문법

조건문

두물정에서는 조건문으로 '만약'을 사용합니다. 이는 다른 언어의 if-else문과 같습니다. 입력으로 숫자형을 받고 입력의 접미사로 '면' 또는 '이면'을 붙입니다. 구문의 마지막에 '한다'를 붙입니다. 입력 값이 0.5 보다 크면 아래있는 구문을 실행합니다. 작으면 '아니면' 구문이 있다면 '아니면' 구문으로 넘기고 없다면 '한다'까지 넘깁니다.

주석 만약 [숫자형 값] 이면 [실행할 구문] 아니면 [실행할 구문] 한다
